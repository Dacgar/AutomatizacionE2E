Instrucciones paso a paso ejecucion de test

1. Descomprimir proyecto
2. Instalar IntelliJ IDEA
3. Dar clic en open
4. Buscar carpeta o directorio donde se descomprimio el proyecto
5. Seleccione la carpeta y de clic en OK
6. Esperar que carguen todas las dependencias
7. Dentro del proyecto, buscar la carpeta src/main/java donde se encontraran los paquetes
8. buscar e ingresar a la carpeta test/java
9. Dentro de la carpeta test/java, buscar el paquete de nombre Runner
10. Abrir la clase de nombre "buyEquipmentRunner", dar clic derecho en la clase TestRunner y seleccionar la opcion "Run buyEquipmentRunner"
11. Esperar que se ejecute el test
12. Al finalizar el test, se mostrara un mensaje en consola con el resultado del test
13. Si el test fue exitoso, se mostrara un mensaje de "Test Passed"
14. Si el test fue fallido, se mostrara un mensaje de "Test Failed"
15. Para visualizar el reporte del test, buscar la carpeta target
16. Dentro de la carpeta target, buscar la carpeta cucumber-reports
17. Dentro de la carpeta cucumber-reports, buscar el archivo de nombre "index.html"
18. Dar clic derecho en el archivo "index.html" y seleccionar la opcion "Open in Browser"
19. Se abrira una ventana en el navegador con el reporte del test
20. En el reporte se mostrara el resultado del test, los escenarios ejecutados, los pasos ejecutados y el tiempo de ejecucion del test
21. Para finalizar, la información que toma la automatización para llenar el formulario de compra se encuentra en el archivo
que se encuentra en la raiz del proyecto, con el nombre "Datos.xlsx"
